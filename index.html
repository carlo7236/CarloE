Exploratory Analysis of Penguins Data Set

The goal of this project was to find a set of 3 variables (2 quantitative, 1 qualitative that were predictive of a penguin's species. With the use of an exploratory analysis, automated feature selection, and 3 different machine learning models we concluded that sex, culmen length, and culmen death were the best predictors of a penguins species.
